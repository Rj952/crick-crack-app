# Crick Crack - Caribbean Literacy App

A world-class literacy app celebrating Caribbean culture through interactive stories, phonics games, and adaptive learning for all ages.

## Features
- Interactive Caribbean Storybook Library
- Phonics & Reading Games with Caribbean vocabulary
- Progress Tracking Dashboard
- Text-to-Speech with Caribbean English
- Adaptive difficulty for all ages

## Tech Stack
- Next.js 14 with App Router
- TypeScript
- Tailwind CSS
- Zustand for state management

## Getting Started
npm install
npm run dev

Open http://localhost:3000

Built with love for Caribbean learners.
